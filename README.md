## Cart Screen

This repository now includes a simple restaurant cart screen demo that can be opened directly in a browser:

- `cart.html` – main HTML page showing selected items, quantity controls, and totals.
- `cart.css` – contains modern, responsive styles for desktop and mobile views.
- `cart.js` – JavaScript powering dynamic quantity updates, removals, and price calculations.

Feel free to integrate these files into the larger application or use them as a starting point for the checkout workflow.
